📌 Variables en Álgebra

Las variables en álgebra representan valores desconocidos y se expresan con letras como x o y.

✨ Ejemplo

Si x + 3 = 7, entonces podemos resolverlo restando 3 en ambos lados:

x = 7 - 3

x = 4

📘 Expresiones Algebraicas

Una expresión algebraica es una combinación de números, variables y operaciones.

Ejemplo

Expresión: 4x + 3

Si x = 5, entonces:

4 × 5 + 3 = 23

📚 Operaciones Básicas con Expresiones

📌 Suma y Resta de Términos Semejantes

Se suman/restan los coeficientes de términos que tienen la misma variable.

✨ Ejemplo de suma

3x + 2x = (3 + 2)x = 5x

✨ Ejemplo de resta

7y − 3y = (7 − 3)y = 4y

📌 Multiplicación de Expresiones Algebraicas

La multiplicación de expresiones algebraicas consiste en multiplicar coeficientes y aplicar las propiedades de los exponentes a las variables. Se deben distribuir los términos correctamente y luego simplificar la expresión.

✨ Ejemplo de Multiplicación

4x × 3x = 12x

📌 División de Expresiones Algebraicas

La división de expresiones algebraicas implica dividir los coeficientes y restar los exponentes de las variables con la misma base. Se debe simplificar la fracción lo más posible.

✨ Ejemplo de División
