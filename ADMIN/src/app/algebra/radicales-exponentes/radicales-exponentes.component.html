📘 Teoría: Exponentes y Radicales

¿Qué es un exponente?

Un exponente indica cuántas veces se multiplica una base por sí misma. Se representa como: aⁿ = a × a × ... × a (n veces).

  • Ejemplo 1: 2³ = 2 × 2 × 2 = 8
  • Ejemplo 2: 5² = 5 × 5 = 25

Propiedades de los exponentes

  • a⁰ = 1 (si a ≠ 0)
  • a¹ = a
  • aⁿ × aᵐ = aⁿ⁺ᵐ
  • (aⁿ)ᵐ = aⁿᵐ
  • a⁻ⁿ = 1 / aⁿ (si a ≠ 0)

¿Qué es un radical?

Un radical representa la raíz de un número. Se usa el símbolo √ para raíces cuadradas, y ∛, ∜ para raíces cúbicas y cuartas.

  • Ejemplo 1: √9 = 3
  • Ejemplo 2: ∛27 = 3
  • Ejemplo 3: √(4²) = 4

Relación entre exponentes y radicales

Los radicales pueden escribirse como exponentes fraccionarios:

  • √a = a^(1/2)
  • ∛a = a^(1/3)
  • ⁿ√a = a^(1/n)
