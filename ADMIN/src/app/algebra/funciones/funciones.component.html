Concepto de Función

Una función es una relación matemática entre dos conjuntos, el **dominio** y el **rango**, en la que a cada elemento del dominio le corresponde un único valor en el rango. Esta relación permite modelar una gran variedad de fenómenos en ciencia, economía, ingeniería y muchas otras disciplinas.

Una función puede representarse de diferentes maneras: mediante expresiones algebraicas, gráficas, tablas de valores o incluso palabras. La característica clave de una función es que no puede haber un mismo valor de entrada con múltiples valores de salida; es decir, para cada x en el dominio, existe un único f(x) en el rango.

Funciones Lineales

Las funciones lineales son aquellas cuya representación gráfica es una **línea recta**. Se expresan mediante la ecuación general:

f(x) = mx + b

  • m: Representa la pendiente de la recta, que indica la inclinación y dirección de la función.
  • b: Es el intercepto en el eje Y, es decir, el punto donde la recta cruza el eje vertical.

Las funciones lineales son ampliamente utilizadas para modelar situaciones de cambio constante, como velocidad uniforme, costos proporcionales o crecimiento lineal.

Ejemplo: Si f(x) = 2x + 3, entonces:

  • Para x = 1, f(1) = 2(1) + 3 = **5**
  • Para x = -2, f(-2) = 2(-2) + 3 = **-1**

Funciones Cuadráticas

Las funciones cuadráticas son aquellas cuya representación gráfica es una **parábola**. Se expresan con la fórmula:

f(x) = ax² + bx + c

  • a: Define la dirección de la parábola; si es positivo, la parábola se abre hacia arriba; si es negativo, hacia abajo.
  • b: Afecta la inclinación y desplazamiento de la parábola en el plano.
  • c: Es el valor donde la parábola intercepta el eje Y.

Este tipo de funciones aparecen en problemas de física, economía y estadística, como el movimiento parabólico de un objeto en caída libre o el análisis de costos en una empresa.

Ejemplo: Si f(x) = x² - 4x + 3, entonces:

  • Para x = 0, f(0) = (0)² - 4(0) + 3 = **3**
  • Para x = 2, f(2) = (2)² - 4(2) + 3 = **-1**

Operaciones Básicas con Funciones

Se pueden realizar diversas operaciones con funciones, lo que permite combinarlas y crear expresiones más complejas. Algunas operaciones fundamentales son:

  • Suma de funciones: (f + g)(x) = f(x) + g(x), combinando los valores de dos funciones.
  • Resta de funciones: (f - g)(x) = f(x) - g(x), obteniendo la diferencia entre dos funciones.
  • Multiplicación de funciones: (f * g)(x) = f(x) * g(x), creando una función más compleja a partir de dos funciones originales.
  • División de funciones: (f / g)(x) = f(x) / g(x), siempre que g(x) ≠ 0.

Ejemplo: Si f(x) = x + 2 y g(x) = x² - 1:

  • Suma: (f + g)(x) = (x + 2) + (x² - 1) = **x² + x + 1**
  • Resta: (f - g)(x) = (x + 2) - (x² - 1) = **-x² + x + 3**
  • Multiplicación: (f * g)(x) = (x + 2) * (x² - 1) = **x³ + 2x² - x - 2**
